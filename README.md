# 🧟 DeadChain Zombie dApp

A basic zombie creation dApp built using Solidity, Hardhat, and Ethers.js.

## 🚀 Features
- Create zombie using smart contract
- Basic frontend interaction
- Quantity selection (UI)

## 🛠 Tech Stack
- Solidity
- Hardhat
- Ethers.js
- HTML, CSS, JavaScript

## 📚 Inspiration / Credits
This project is inspired by the CryptoZombies course by Loom Network.  
This project is built for learning purposes only.

## ⚠️ Note
This is not a full game. It only demonstrates basic smart contract interaction.
